# Glider Simulation Web

This is a browser-based glider simulation built for quick sharing and demo use. The Hornsea 2 Project wind farm scene is reconstructed from real turbine coordinates and seafloor terrain data, so the environment matches the real-world layout as closely as possible.

Live Demo: : https://shaco-f.github.io/Glider_simulation_web/
