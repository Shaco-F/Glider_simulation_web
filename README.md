# Glider Simulation Web

A browser-based interactive underwater glider simulation designed for quick demonstration and sharing.  
The Hornsea 2 offshore wind farm environment is reconstructed using real turbine coordinates and seabed terrain data to provide a realistic operational scene.

## Features

• Real-world Hornsea 2 wind farm reconstruction  
• Interactive underwater glider visualization  
• Real-time camera navigation  
• Monitoring panel for glider tracking  
• Lightweight WebGL deployment for easy sharing  

---

## Controls

### Keyboard Controls

| Key | Function |
|-----|----------|
| **W** | Move forward |
| **S** | Move backward |
| **A** | Move left |
| **D** | Move right |
| **Space** | Move up |
| **Alt** | Move down |
| **Tab** | Change type of map |

### Mouse Controls

| Action | Function |
|--------|----------|
| **Right click + drag** | Rotate camera view |
| **Left click** | Interaction |
| **Click monitor panel** | Track glider camera |

---

## Usage Tips

• Click the monitoring panel to switch to glider tracking mode  
• Use free camera mode to explore the wind farm layout  
• Combine keyboard movement and mouse rotation for smooth navigation
